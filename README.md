A real-time cloud security monitoring dashboard simulating an AWS Security Operations Center. Built with vanilla HTML, CSS, and JavaScript — no dependencies.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| **Live Threat Feed** | Real-time security event stream with severity classification |
| **World Threat Map** | Animated visualization of attack origins across global regions |
| **IAM Activity Chart** | 7-day bar chart of authentication attempts |
| **AWS Region Health** | Per-region security and uptime scores |
| **Compliance Scoring** | SOC 2 / ISO 27001 aligned metrics |
| **Critical Alerts** | Dismissible notification banner system |

## 🛡 Security Concepts Demonstrated

- IAM (Identity & Access Management) monitoring
- Multi-Region health visualization (EC2, S3, IAM)
- Threat classification — CVSS-style severity tagging
- MFA enforcement tracking
- S3 data egress anomaly detection
- Brute force / port scan event logging
